# hello git

## git 명령어 요약

 - clone : 원격 저장소 복사
 - add : 스테이지 영역에 작업 파일 추가
 - commit : 세이브, 스테이지 영역의 파일들을 가지고 세이브 하는 방식
 - push : commit된 내용 원격 저장소에 업로드

 ## 브랜치
  - 브랜치랑 : 기존 내용을 유지한대로 새로운 내용을 추가하고 싶을때 사용한다.
  - 체크아웃 : 특정 브랜치(혹은 커밋)으로 돌아가고 싶을 때 사용.
  - 소스트리의 체크아웃:브랜치 이름을 더블 클릭하는 것만으로 체크아웃 가능.
